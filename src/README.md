# Interactive Library Management System (CLI)
An interactive, console-based Java application designed to manage a library's books and users. This project demonstrates strong Object-Oriented Programming (OOP) principles and robust error handling.

## Features
- **Dynamic Command Line Interface (CLI):** A persistent menu loop that allows users to interact with the system in real-time.

- **User Management:** Register both Student and Teacher users using polymorphism.

- **Inventory Control:** Add books with specific Enums for categories (Action, Tech, Romance, etc.).

- **Borrowing Logic:** A search-based system that allows users to borrow or return books using specific IDs and Titles.

- **Input Validation:** Custom getSafeInt logic using try-catch to ensure the program never crashes on invalid user input.

## Technical Stack
- **Language:** Java

- **Concepts:** Inheritance, Polymorphism, Enums, ArrayLists, and Exception Handling.

- **Structure:** Organized using a clean package architecture (models, services, main).

## How to Use
1. Open the project in an IDE (like IntelliJ IDEA).
2. Navigate to src/Main.java.
3. Run the main method.
4. Follow the on-screen prompts to manage your library!
