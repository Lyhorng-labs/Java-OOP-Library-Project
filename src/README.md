# My First Library Management System

A Java-based console application designed to manage a library inventory and user borrowing system. This project serves as a foundational demonstration of **Object-Oriented Programming (OOP)**

## Features
- models.Book Management: Add books with titles, authors, and categories.
- models.User System: Register users, track their personal borrowed lists.
- Search System: Find books by title.
- Return System: Handled by the Library to update book availability.

## How to Run
- Open the project in an IDE (like IntelliJ).
- Run the `Main.java` file.
- View the simulation output in the console.

## Learning Objectives
- **OOP Architecture:** Coordinating interactions between multiple classes.
- **Encapsulation:** Using private fields and public methods to protect data.
- **Collections:** Managing and searching dynamic data with `ArrayList`.
- **Logic Design:** Implementing checks for book availability and search results.
